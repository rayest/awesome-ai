## 使用建议

1. **生成顺序**：先做 #1、#4、#9、#11（四张比喻图）→ 适合做封面
2. **技术架构**：#3、#5、#6、#7、#10 → 公众号正文分段配图
3. **汇总页**：#8、#12 → 适合做结尾总结图 / 朋友圈分享卡
4. **风格统一**：所有提示词都强调 "clean infographic / minimalist / editorial"，保持视觉一致性
5. **二次精修**：AI 出图后可用 PS/Canva 调整中文标签的字体与位置，确保排版精致
6. **URL 编码**：实际调用时需将 prompt 进行 URL 编码（如空格变 `%20`、中文需 UTF-8 编码）
