# Image Replicator（图片复刻提示词生成器）

## Purpose

把用户提供的**图片**（或图片描述）转化为**可重新生成的英文 prompt**，核心要求：

1. **精确识别视觉内容**：布局 / 物体 / 人物 / 配色 / 风格 / 构图 / 光线
2. **保留核心视觉元素**：原图的主体结构、关键物体位置、视觉风格不能丢
3. **重新组织文字内容**：原图所有文字（标题 / 标签 / 注释 / 数字）必须**重写而非照抄**
4. **同图不同字**：视觉等价、文字原创

适用场景：竞品海报二次创作 / 同款课程封面改版 / 信息图风格迁移 / 避免版权文字复制 / 跨语言复用。

## Trigger

用户提供图片 + 复刻意图时立即触发：

| 用户动作 | 触发命令 |
|----------|----------|
| 上传图片 | "复刻这张" / "做一张类似的" / "同款换文字" |
| 引用图片 URL | "重新生成这个风格" / "改一下文字保留风格" |
| 描述图片 | "这是 X 风格的海报，复刻一份" |
| 粘贴图片分析 | "按这个分析重新做" |

> 用户没说保留哪些 / 改哪些时，**默认** = 保留全部视觉元素 + 改写全部文字。

## Workflow

### 1. 解析输入

| 要素 | 来源 | 可覆盖 |
|------|------|--------|
| 图片 | 用户上传 / URL / 用户描述 | 无 |
| 复刻精度 | 默认"高保真"（90%+ 视觉相似） | 用户指定（粗糙复刻 / 风格借鉴 / 灵感参考） |
| 文字策略 | 默认"完全重写"（避免照抄） | 用户指定（仅改主标题 / 改标签保标题 / 改所有文字） |
| 文字语言 | 默认中文 | 用户指定（英文 / 双语 / 原文同语种重写） |
| 输出尺寸 | 默认 landscape_16_9 | 用户指定 |
| 输出用途 | 默认通用 | 用户指定（公众号 / 课程 / 海报 / 配图） |

### 2. 视觉识别（X 光扫描）

**把图片拆成 7 个视觉维度**（不展示用户，仅作内部加工）：

| 维度 | 提取内容 |
|------|----------|
| **布局** | 单图 / 多宫格 / 时间线 / 矩阵 / 圆环 / 流程图 / 对比图 / 信息图 |
| **主体** | 中心物体 / 人物 / 场景 / 数据图表 / 装饰元素 |
| **配色** | 主色 + 辅色 + 点缀色（精确到色号或近似色名） |
| **字体** | 黑体 / 宋体 / 手写体 / 衬线 / 等宽 + 字重 + 大小关系 |
| **风格** | 写实 / 扁平 / 拟物 / 插画 / 3D / 等距 / 中国风 / 赛博朋克 / 极简 |
| **构图** | 中心对称 / 三分法 / 黄金螺旋 / 上下结构 / 网格 / 留白比例 |
| **光线 / 材质** | 暖光 / 冷光 / 高光 / 阴影 / 哑光 / 金属 / 玻璃 / 木质 |

### 3. 文字提取与重写

**关键约束**：原图所有文字（标题 / 副标题 / 标签 / 注释 / 数字 / 角标 / 水印）**不能原样照搬**。

**重写策略**（5 选 1，根据场景决定）：

| 策略 | 用法 | 适用场景 |
|------|------|----------|
| **同义改写** | "AI 时代" → "智能革命" | 简短标题 |
| **结构重组** | "5 个方法" → "3 步搞定" | 信息图标签 |
| **角度转换** | "如何减肥" → "减脂的科学路径" | 提问式标题 |
| **数字重置** | "7 步法" → "完整流程" | 含数字的标题 |
| **跨语种重写** | 中文标题 → 英文（再译回中文） | 完全规避原文 |

**文字数量约束**：
- 标题：原文字数 ±30%
- 副标题：原文字数 ±20%
- 标签 / 注释：可完全重写，字数大幅压缩
- 数字：可保留（如"99%"）或换算（如"绝大多数"），**不能原样照抄连续 5 字以上**

### 4. 风格映射

**把视觉风格翻译成英文 prompt 关键词**：

| 中文风格 | 英文 prompt 关键词 |
|----------|-------------------|
| 极简 | minimal, clean, flat, whitespace, geometric |
| 写实 | photorealistic, cinematic, ultra-detailed, natural lighting |
| 插画 | illustrated, hand-drawn, painterly, soft edges |
| 3D | 3D render, isometric, volumetric, ray tracing |
| 中国风 | traditional Chinese, ink wash, calligraphic, red and gold |
| 赛博朋克 | cyberpunk, neon, holographic, glitch, dark futuristic |
| 信息图 | infographic, data visualization, chart, diagram, annotated |
| 商务 | corporate, professional, navy blue, geometric, sans-serif |

### 5. Prompt 拼接

**最终 prompt 结构**（英文）：

```
A [风格] [尺寸] illustration of [主体描述], featuring [核心元素 1], [核心元素 2], ... The composition uses [布局 / 构图], with [配色] color palette. The text elements include "[改写后的主标题]", "[改写后的副标题]", and labels for [改写后的标签 1] / [改写后的标签 2] (in [字体风格]). Style: [风格关键词], [光线关键词], [材质关键词], 16:9 wide composition.
```

**示例**：

原图：一张"AI 改变未来"风格海报，蓝色渐变背景，中心 AI 机器人大脑图标，下方"5 大趋势"标签列表

输出 prompt：
```
A cinematic wide-angle digital illustration of an AI neural network with a glowing central node radiating electric blue connections to five surrounding concept icons, set against a deep navy gradient background with subtle particle effects. The composition is centered and symmetrical, with the title "智能浪潮：未来已来" in bold modern sans-serif at the top, the subtitle "重塑每一个行业" in lighter weight below, and five floating labels reading "基础设施" / "创作革命" / "决策升级" / "效率倍增" / "体验重塑" (in light cyan). Color palette: deep navy (#0A1929) background, electric cyan (#00D9FF) accents, gold highlights (#FFD700). Style: cyberpunk editorial, neon glow, ultra-detailed, 4K render. 16:9 wide composition.
```

### 6. 输出规范

**不写文件**（除非用户明确说"保存到文件"），**直接在对话中输出**。

**输出结构**：

```markdown

- 布局：[布局类型]
- 主体：[主体描述]

- 风格：[风格]
- 构图：[构图方式]
- 文字数量：[N 处标题 + M 个标签 + K 个注释]

**文字重写对照表**（展示原文与重写后对比，让用户验证）：

| 位置 | 原文 | 重写后 | 策略 |
|------|------|--------|------|
| 主标题 | AI 改变未来 | 智能浪潮：未来已来 | 同义改写 |
| 副标题 | 5 大趋势 | 重塑每一个行业 | 角度转换 |
| 标签 1 | 自动化 | 效率倍增 | 同义改写 |
| ... | ... | ... | ... |

**生成的英文 Prompt**：
```


## Rules

- **绝不**原样照搬原图文字（即使只 1 个字也不能连续 5 字以上相同）
- **绝不**丢失原图核心视觉元素（主体 / 布局 / 风格必须保留）
- **绝不**把"复刻"理解成"用同款 prompt 模板"（必须基于原图实际分析）
- **绝不**在 prompt 中包含"复刻 / 模仿 / 类似 / copy / replicate"等元描述词
- **绝不**省略文字重写对照表（用户要能验证文字没照抄）
- **绝不**在 prompt 中暴露原图来源（"inspired by..."、原作者信息等）
- **绝不**使用原图的版权 / 商标 / 品牌 logo（必要时替换为占位符）
- **绝不**改变原图的核心信息密度（不能为改而删）
- 用户没说英文就**默认中文**输出
- 用户没说输出尺寸就**默认** `landscape_16_9`

## Anti-patterns

- ❌ 不要把"复刻"做成"风格借鉴"（视觉元素必须 1:1 对应）
- ❌ 不要文字重写后语义大变（原标题"AI 入门指南"不能改成"AI 进阶秘籍"）
- ❌ 不要把多语言文字直接翻译而不重组（必须结构 + 表达都重写）
- ❌ 不要在 prompt 里堆砌 50 个关键词（聚焦 10-15 个最关键的）
- ❌ 不要把原图所有细节都搬到 prompt（保留核心，省略冗余）
- ❌ 不要忽略原图的光线 / 材质细节（这是风格的核心）
- ❌ 不要做"差不多的复刻"（必须视觉高度一致，仅文字不同）

## Quick Reference（速查）

| 图片类型 | 复刻策略 | 重点保留 |
|----------|----------|----------|
| 海报 / 封面 | 整体布局 + 主视觉 + 配色 | 中心元素 + 标题位置 |
| 信息图 | 数据结构 + 模块划分 | 图表类型 + 标签位置 |
| 课程图 | 章节结构 + 视觉锚点 | 流程 + 关键节点 |
| 截图 / UI | 界面元素 + 文字框 | 按钮 + 字段 + 状态 |
| 摄影 / 写实 | 场景 + 人物 + 光线 | 主体 + 背景 + 色调 |
| 插画 / 3D | 角色 + 动作 + 风格 | 角色特征 + 风格关键词 |

| 文字重写策略 | 触发条件 | 示例 |
|-------------|----------|------|
| 同义改写 | 标题简短（<8 字） | "AI 入门" → "智能启蒙" |
| 结构重组 | 含数字 / 列表 | "5 大趋势" → "未来已来" |
| 角度转换 | 提问 / 命令式 | "如何减肥" → "减脂的科学" |
| 数字重置 | 含具体数字 | "100 种方法" → "完整方案" |
| 跨语种重写 | 完全规避原文 | 中 → 英 → 中（重新表述） |

## 依赖

- 视觉能力（模型自带 vision 或用户描述）
- 联网（如需补图原信息 / 风格参考）
- 无本地依赖

## 示例对话

**用户**：
```
[上传一张"AI Agent 改变未来"风格的公众号封面]

复刻这张图，文字全改。
```

**本 Skill 自动执行**：
1. 视觉识别：公众号封面 / 横版 / 深蓝渐变背景 / 中心 AI 大脑图标 / 5 个标签放射状排列 / 主标题 + 副标题 + 5 标签
2. 文字提取：主标题"AI Agent 改变未来" / 副标题"5 大趋势" / 标签：自动化 / 创作 / 决策 / 效率 / 体验
3. 文字重写：
   - 主标题："AI Agent 改变未来" → "智能体浪潮：未来已来"
   - 副标题："5 大趋势" → "重塑每一个行业"
   - 标签：自动化→效率倍增 / 创作→创作革命 / 决策→决策升级 / 效率→效率倍增（避免重复）/ 体验→体验重塑
4. Prompt 拼接：英文 prompt 包含风格 + 主体 + 配色 + 改写后文字
5. 输出：复刻 prompt + 文字对照表

**用户**：
```
[上传一张"3 步学会 X"风格的步骤图]

复刻成 5 步的"3 步学会 Y"。
```

**本 Skill 自动执行**：
1. 视觉识别：横版步骤图 / 浅色背景 / 3 个圆圈数字 + 文字 + 箭头 / 极简风
2. 视觉保留：圆圈 + 数字 + 箭头 + 极简风
3. 视觉调整：3 个圆圈 → 5 个圆圈
4. 文字重写：3 步学会 X → 3 步学会 Y（保持"X 步学会"结构）
5. 输出：复刻 prompt

**用户**：
```
[上传一张信息图：2025 年 AI 行业 6 大数据]

复刻，但数据全换。
```

**本 Skill 自动执行**：
1. 视觉识别：信息图 / 多模块 / 6 个数据卡片 / 圆环 / 进度条 / 数字大字号
2. 视觉保留：6 卡片布局 + 数字大字号 + 配色
3. 视觉调整：6 个数据 → 6 个新数据（可联网补 2025 年新数据）
4. 文字重写：所有标题、标签、注释全重写
5. 输出：复刻 prompt

**用户**：
```
[粘贴一段图片分析描述]

按这个分析复刻。
```

**本 Skill 自动执行**：
1. 解析：用户已提供图片分析（视觉 + 文字）
2. 直接基于描述生成 prompt
3. 文字重写：基于分析中的文字部分
4. 输出：复刻 prompt

**用户**：
```
[上传一张"AI 提示词 6 大原则"配图]

复刻成"AI 提示词 8 大原则"。
```

**本 Skill 自动执行**：
1. 视觉识别：横版 / 信息图 / 6 个原则卡片网格排列 / 图标 + 标题 + 描述 / 浅色 + 蓝绿配色
2. 视觉保留：网格布局 + 图标 + 标题 + 描述 + 配色
3. 视觉调整：6 卡片 → 8 卡片（2 行 4 列 → 2 行 4 列 或 4 行 2 列）
4. 文字重写：6 大原则 → 8 大原则，原 6 个原则名 + 描述全重写
5. 联网（如有需要）：补 2 个新原则
6. 输出：复刻 prompt
