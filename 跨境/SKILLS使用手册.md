## 1. ecommerce-research — 选品与市场调研

**文件路径：** [skills/ecommerce-research/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-research/SKILL.md)

### 功能介绍

这是整条供应链的第一道关卡。它提供一套**端到端的选品决策框架**，从市场机会识别到选品最终拍板，所有环节都有对应的分析模板和评分标准。它不会给你模糊的"可以做/不建议做"，而是强制要求输出 **GO / TEST / NO-GO** 的明确结论。

### 能帮你做什么

- **市场趋势判断**：帮你分析某个品类的搜索量趋势、市场增长率，判断这是蓝海还是红海
- **竞品分析**：输出结构化的竞品档案，包括价格区间、月销量估算、好评/差评关键词以及你可以切入的差异化机会
- **选品评分卡**：对候选产品从市场需求、竞争强度、利润空间、供应链难度、物流适配、合规风险六个维度打分（1-5分），自动算出加权总分
- **供应链初筛**：提供 1688/阿里巴巴 搜索方向、产业带定位建议、认证要求清单
- **定价策略**：自动套入跨境或国内的定价公式，给出合理的定价区间和策略建议
- **选品决策报告**：输出包含执行摘要、竞品汇总、风险点和下一步行动的完整报告

### 覆盖平台

Amazon、Shopee、Lazada、Temu、TikTok Shop、独立站（Shopify等）、淘宝、京东、抖音、拼多多

### 使用示例

```
"帮我做一个适合美国亚马逊站的宠物用品选品分析，预算5万人民币首批"
"分析一下抖音电商的家居收纳类目，有没有适合切入的机会"
"这个无线蓝牙耳机的竞品差评都在骂什么，我能从哪里差异化"
```

### 需要你提供的信息

| 信息项 | 为什么要问 |
|--------|-----------|
| 目标平台 | 不同平台调研逻辑完全不同 |
| 目标品类 | 避免分析方向跑偏 |
| 采购预算 | 影响 MOQ 和供应商选择 |
| 风险偏好 | 决定选品策略（跟卖/测试/创新） |

> **注意**：此 skill 无法实时联网抓取 Jungle Scout 或卖家精灵的数据，分析基于训练数据和你提供的信息。关键数据建议用专业工具交叉核实。

## 2. ecommerce-listing — 商品上架与内容优化

**文件路径：** [skills/ecommerce-listing/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-listing/SKILL.md)

### 功能介绍

Listing 是电商产品的"门面"，直接决定点击率和转化率。这个 skill 提供**平台专属**的 Listing 创作和优化流程，从关键词布局到多语言生成一气呵成，每个环节都有严格的平台规则约束，避免因违规而导致 Listing 被压制。

### 能帮你做什么

- **关键词布局规划**：设计哪些词放标题（大词/核心词），哪些词放五点描述（长尾词），哪些词补充到 Search Terms 后台
- **标题创作**：按照各平台公式和字符限制规范生成标题
  - Amazon：< 200字符，禁止促销词，核心词前置
  - 淘宝：30个汉字（60字节）优化黄金法则
  - Shopee：< 120字符，移动端友好
- **五点描述（Bullet Points）创作**：采用`【大写卖点标题】 + 展开说明 + 数据支撑`的专业结构，而不是流水账
- **详情页/A+页面规划**：输出内容模块方案，规定哪个模块放什么内容（品牌故事、功能详解、规格参数、使用场景等）
- **视觉素材 Brief**：告诉你摄影/设计团队每张图的要求（尺寸、构图、内容重点）
- **多语言版本**：在英文母版完成后，生成德语、日语、法语、西班牙语等版本，并处理文化适配
- **合规检查**：发布前自动过滤绝对化用语（最好、第一、100%等）、虚假宣传、平台违禁词

### 覆盖平台

Amazon（美/欧/日）、Shopee/Lazada、Temu、淘宝/天猫、京东、抖音电商、拼多多、独立站

### 使用示例

```
"帮我写一个无线蓝牙耳机的亚马逊美国站完整 Listing，它的核心卖点是主动降噪和40小时续航"
"把这个中文淘宝标题翻译并优化成适合亚马逊日本站的版本"
"帮我设计这款咖啡机的 A+ 内容模块方案"
```

### 关键平台规则速查（内置于 skill）

| 平台 | 标题限制 | 关键词后台 | 特殊规则 |
|------|---------|-----------|---------|
| Amazon | < 200字符 | 249字节 | 禁促销词、禁全大写 |
| 淘宝 | 30个汉字 | 无专门后台 | 填满为优 |
| Shopee | < 120字符 | 无 | 移动端截断注意 |
| 抖音 | 短平快 | 无 | 3秒内抓注意力 |

## 3. ecommerce-advertising — 广告投放与营销

**文件路径：** [skills/ecommerce-advertising/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-advertising/SKILL.md)

### 功能介绍

广告是电商流量的发动机。这个 skill 覆盖**跨境与国内主流平台**的全套广告打法，从策略制定到日常 SOP，从预算分配到素材创意，每个环节都有标准流程和数据决策框架。

### 能帮你做什么

- **Amazon PPC 投放结构搭建**：设计三层金字塔结构（自动广告探词 → 精准广告转化 → 广泛广告扩量），明确每层预算占比和竞价策略
- **ACOS 目标设定**：根据你的毛利率和销售目标，帮你算出合理的目标 ACOS（而不是拍脑袋定 15%）
- **Facebook/Instagram 漏斗投放**：设计冷流量（兴趣定向/Lookalike）、温流量（网站访客再营销）、热流量（加购未付款）三层受众策略
- **抖音巨量千川投放**：规划达人定向、行为兴趣定向、DMP人群包的使用逻辑和素材创意方向
- **淘宝直通车/引力魔方**：关键词布局、人群溢价、时间折扣的策略框架
- **素材创意脚本**：提供痛点型（15秒）、教育型（30秒）等多种脚本模板，并给出各平台素材规格速查
- **预算分配模型**：新品期和成熟期各自的预算分配比例建议
- **数据监控 SOP**：每日/每周/每月应该看哪些指标、怎么决策
- **月度投放效果报告**：按模板输出分渠道分析和下月计划

### 覆盖平台

Amazon PPC（SP/SB/SD）、Facebook/Instagram Ads、Google Ads、TikTok Ads、巨量千川、淘宝直通车/引力魔方、京东广告

### 使用示例

```
"我新品刚上 Amazon，预算每天$100，帮我设计广告结构"
"我的 ACOS 已经 35% 了，毛利只有 28%，帮我分析该怎么调"
"帮我写一个15秒的千川短视频广告脚本，产品是电动扭蛋机，受众是25-35岁的年轻妈妈"
"我的 Facebook 广告上周 ROAS 从 3.2 掉到 1.8，帮我分析可能的原因和排查步骤"
```

### 核心指标健康阈值（内置于 skill）

| 指标 | 含义 | 健康值 |
|------|------|--------|
| CTR | 点击率 | > 0.3%（展示型）/ > 1%（搜索型） |
| CVR | 转化率 | > 5% |
| ACOS | 广告花费/广告销售额 | < 毛利率 |
| ROAS | 销售额/花费 | > 3（独立站）/ > 2（平台） |

## 4. ecommerce-procurement — 采购与供应商管理

**文件路径：** [skills/ecommerce-procurement/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-procurement/SKILL.md)

### 功能介绍

采购直接决定产品质量和成本结构。这个 skill 提供从**供应商开发到绩效考核**的全生命周期管理框架，并配有量化评分工具，避免凭感觉选供应商。

### 能帮你做什么

- **供应商初筛**：生成结构化的初筛表（资质核验、MOQ、交期、样品政策），快速排除不合格供应商
- **现场验厂检查表**：列出生产设施、质量管理体系、合规认证、商务条件四个维度的评估项，可直接打印带去工厂
- **供应商评分卡**：从生产能力（20%）、品质管理（25%）、认证合规（15%）、商务条件（20%）、沟通响应（20%）五维度打分，自动输出 A/B/C/D 级别
- **采购订单模板（PO单）**：标准格式的采购订单，含商品明细、质量要求、物流要求
- **三级质检体系**：产前样（Pre-production Sample）→ 中期验货 → 尾期验货（FRI），提供每个阶段的抽样标准（AQL 1.5/2.5）
- **供应商绩效月报**：交付表现（准时交货率/完整率）、质量表现（来料合格率/退货率）、服务表现的定期考核表
- **成本拆解模型**：把采购成本拆解到原材料、加工费、包装费、模具分摊、认证分摊等层级，精确核算真实成本
- **账期谈判策略**：按合作阶段给出谈判筹码和目标账期建议

### 使用示例

```
"帮我列一份准备和工厂谈判的供应商评估清单"
"有3家供应商，帮我设计一个评分表来选出最优的"
"帮我算一下这个SKU的真实采购成本，工厂报价8元，MOQ是1000件"
"帮我起草一封催促供应商提交产前样的邮件（中英双语）"
```

## 5. ecommerce-warehouse — 仓储与库存管理

**文件路径：** [skills/ecommerce-warehouse/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-warehouse/SKILL.md)

### 功能介绍

库存是电商最大的资金占用来源，管不好就会滞销积压或缺货断货。这个 skill 提供**科学的库存控制方法论**，内置安全库存、补货点、经济订货量（EOQ）的精确计算公式，让补货决策有据可依而不是凭直觉。

### 能帮你做什么

- **ABC 分类分析**：按销量占比把 SKU 分为 A（重点）/B（一般）/C（淘汰候选）三类，制定差异化管理策略
- **安全库存计算**：根据日销量波动（标准差）和补货周期，算出统计学意义上的安全库存数量（支持 95%/99% 服务水平）
- **补货点与 EOQ 计算**：精确计算什么时候该下补货单、每次补多少量
- **库存预警规则设计**：帮你定义缺货预警（可售天数≤3天）、低库存（7天）、滞销（库龄>90天）、超储的触发条件和处理时限
- **库存健康度仪表盘模板**：整体概况 + ABC分布 + 预警清单一张表
- **滞销品处理决策树**：根据库龄（90天/180天/365天）自动推荐处理策略（降价促销 → 站外清仓 → 报废捐赠）
- **多仓库存分配方案**：根据各仓库的销量占比、物流成本、时效优势计算最优分货比例
- **周转率提升策略**：精准预测、小批量多频次、预售模式、供应商 VMI 等策略及其预期效果

### 覆盖场景

国内（平台官方仓/第三方仓/自建仓）、跨境（FBA/海外仓/保税仓/自发货）

### 使用示例

```
"我有一个SKU，日均销量20件，标准差5件，补货周期30天，帮我算安全库存和补货点"
"帮我对下面这批SKU做ABC分类，顺便给出每类的管理建议"
"我有一批家居用品在仓库已经堆了120天了，帮我出一个清仓方案"
"FBA仓库的IPI分快到最低线了，帮我出一个库存瘦身计划"
```

## 6. ecommerce-logistics — 物流与履约

**文件路径：** [skills/ecommerce-logistics/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-logistics/SKILL.md)

### 功能介绍

物流是电商履约体验的核心，也是成本结构中最难控制的一块。这个 skill 帮你**系统设计物流方案**，对比不同方案的成本、时效、风险，给出可量化的选择建议，而不是拍脑袋定。

### 能帮你做什么

- **物流模式对比与选择**：国内（通达系/顺丰/京东物流）和跨境（FBA/海外仓/自发货专线/邮政小包）各方案的时效、成本、适用场景对比
- **头程物流方案规划**：海运整柜/拼箱/空运/中欧班列/国际快递的详细对比（时效从3天到50天，成本从每公斤$2到$15）
- **头程全成本核算表**：运输费 + 报关费 + 文件费 + 保险费 + 目的港费用的完整汇总，精确到单件分摊
- **尾程配送网络设计**：本地邮政/商业快递/平台物流的覆盖范围和适用场景分析
- **智能路由推荐**：按照时效要求（30%权重）、成本（25%）、商品属性（20%）、目的地（15%）、服务等级（10%）的加权模型推荐最优方案
- **物流异常处理 SOP**：未揽收（48小时）、运输停滞（72小时）、清关延误、包裹丢失、破损各类情形的标准处理流程和时效要求
- **物流 KPI 体系**：准时送达率（≥95%）、物流成本占比（国内<8%/跨境<15%）、破损丢失率（≤0.5%）等核心指标健康阈值

### 使用示例

```
"我有一批500kg的蓝牙耳机要从深圳发到美国亚马逊仓，海运和空运各多少钱，哪个合适"
"帮我出一份从中国发货到德国的头程成本核算，产品是折叠椅，40个/箱，共50箱"
"我的 FBA 库存快没了，等不了海运，现在空运一批过去紧急补货，帮我计算一下成本"
"Shopee 马来西亚站用哪家物流商比较好？时效和价格分别是多少？"
```

## 7. ecommerce-customer-service — 客服与售后

**文件路径：** [skills/ecommerce-customer-service/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-customer-service/SKILL.md)

### 功能介绍

客服质量直接影响店铺评分、账号健康和复购率。这个 skill 最大的亮点是**多语言支持**（中/英/德/日/法/西班牙）和覆盖**国内+跨境**两套完整话术体系，从售前咨询到差评处理全场景覆盖。

### 能帮你做什么

- **售前话术生成**：针对产品咨询、价格谈判、物流时效咨询等场景，直接生成中英双语的专业话术（国内风格亲切、跨境风格专业）
- **订单全流程消息模板**：催付提醒、发货通知、签收关怀各节点的标准消息
- **物流异常安抚话术**：针对物流停滞、包裹丢失、破损等情形的中英双语应急话术

## 8. ecommerce-crm — 客户关系与私域运营

**文件路径：** [skills/ecommerce-crm/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-crm/SKILL.md)

### 功能介绍

老客户的价值远被低估。这个 skill 提供**科学的客户分层方法（RFM 模型）**和系统化的私域运营框架，帮你把流量转化为资产，把一次性买家培养成长期忠实客户，提升客户生命周期价值（LTV）。

### 能帮你做什么

- **客户画像构建**：按交易特征（消费金额/订单数/客单价）、行为特征（活跃时间/促销敏感度）输出结构化客户档案
- **RFM 分层分析**：按最近购买时间（R）、购买频率（F）、消费金额（M）三维打分，把客户分为核心VIP/重要客户/潜力客户/普通客户/流失客户5层，并给出每层的差异化运营策略
- **会员体系设计**：设计普通/银卡/金卡/白金/黑卡的五级会员权益体系、成长值规则和积分体系
- **企业微信（企微）运营 SOP**：加粉钩子设计、标签体系搭建、内容节奏（周一新品预告、周三种草、周五福利、周日互动）、转化路径设计
- **社群运营框架**：VIP专属群/品类兴趣群/活动快闪群/老客福利群各自的人数上限、运营目标和内容节奏
- **复购激励方案**：复购券、会员日、积分加速、订阅制、补货提醒各策略的预期效果（复购率+15%~+25%）
- **流失预警模型**：黄色（30天）/橙色（60天）/红色（90天）三级预警机制和配套挽回话术
- **营销自动化场景**：首单感谢、加购未付款提醒、签收邀评、生日礼、沉睡召回等7个自动化场景的触发条件和推送渠道
- **LTV 计算公式**：`LTV = 客单价 × 购买频率 × 客户生命周期 × 毛利率`，并给出分层提升策略

### 使用示例

```
"帮我设计一套淘宝店铺的会员体系，分5个等级，核心客单价在200-500元区间"
"给我出一份企微私域的运营 SOP，我们是做女性护肤品的"
"下面这批客户数据，帮我做 RFM 分层，并给出每层的运营建议"
"帮我写一条30天未购买客户的挽回短信，产品是儿童益智玩具"
```

## 9. ecommerce-analytics — 数据分析与报表

**文件路径：** [skills/ecommerce-analytics/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-analytics/SKILL.md)

### 功能介绍

"感觉今天销量不太好"是经营大忌，数字才是真相。这个 skill 建立了一套从**基础指标到深度诊断**的完整数据分析框架，覆盖日报、周报、月报全周期，并内置跨境和国内两套利润核算公式。

### 能帮你做什么

- **核心指标金字塔体系**：从业绩指标（GMV/订单量）→ 流量指标（CTR/CPC）→ 转化指标（CVR/加购率）→ 盈利指标（毛利率/ROAS）→ 运营效率（库存周转/退货率）五层完整拆解
- **日报/周报/月报模板**：提供标准化的报表结构，包含今日 vs 昨日 vs 上周同日的对比，流量来源分析，广告表现，异动说明
- **单品利润核算模型**：跨境版（含头程分摊、FBA费用、广告分摊、退货损耗）和国内版（含快递费、平台扣点、推广费、人工分摊）各自的完整核算表
- **库存健康分析报告**：整体概况 + ABC分类分析 + 滞销品处理建议 + 补货清单
- **竞品监控报告**：定期追踪竞品价格、评分、Reviews、排名、促销活动的变化，并给出市场机会识别和应对策略
- **数据可视化指导**：告诉你什么数据该用折线图/柱状图/漏斗图/饼图呈现
- **Python 自动化报表框架**：提供 `auto_report.py` 脚本结构，支持从平台 API 数据生成日报/利润报表

### 使用示例

```
"帮我生成这周的运营周报模板，我来填数据，然后你帮我分析"
"今天的转化率从3.2%跌到1.8%，流量没变，帮我分析可能的原因和排查步骤"
"帮我核算这个SKU的真实净利润：售价$29.99，采购成本¥35，头程分摊$1.5，FBA费用$6，ACOS20%，退货率8%"
"帮我建一套适合亚马逊卖家的每日运营检查清单"
```

## 10. ecommerce-bi — 商业智能与数据看板

**文件路径：** [skills/ecommerce-bi/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-bi/SKILL.md)

### 功能介绍

当你的业务从一个平台扩展到多个平台，数据就会变得四分五裂、口径混乱。这个 skill 提供**多平台数据整合（ETL）框架和 BI 看板体系**，帮你把淘宝、京东、抖音、Amazon 的数据统一到一套标准，真正实现数据驱动决策。

### 能帮你做什么

- **数据源矩阵规划**：告诉你淘宝（生意参谋API）、京东（京准通API）、抖音（抖店API）、Amazon（SP-API）各自的数据接入方式和更新频率
- **数据接入方案对比**：API直连 vs ERP中转 vs RPA采集 vs 第三方数据服务四种方案的优劣势分析
- **ETL 标准化映射**：提供各平台"订单ID/订单金额/下单时间/商品SKU/物流状态"字段的标准映射表，解决跨平台数据口径不一致问题
- **ETL 流程代码框架**：提供 Python 的 `extract → transform → load` 标准代码结构
- **运营驾驶舱看板设计**：顶部核心 KPI（GMV/订单量/客单价/转化率带🟢🟡🔴状态）+ 中部趋势图 + 底部异常预警
- **销售预测模型**：移动平均、指数平滑、ARIMA 的适用场景和 Python 代码实现
- **报表体系设计**：运营日报（每日）/ 周报（每周）/ 财务月报（每月）/ 库存周报 / 竞品月报的自动化推送规则设计
- **数据治理标准**：完整性（关键字段非空率>99%）/准确性/一致性/及时性（延迟<2小时）/唯一性五项数据质量标准

"帮我用指数平滑法预测未来7天的销量，历史数据如下：..."

## 11. ecommerce-compliance — 合规、税务与风控

**文件路径：** [skills/ecommerce-compliance/SKILL.md](file:///Volumes/BIWIN_DEV/跨境/skills/ecommerce-compliance/SKILL.md)

### 功能介绍

合规失误是跨境电商最高风险的地雷，一个侵权或税务问题就能让店铺被封、资金被冻结。这个 skill 提供**针对主要目标市场的合规检查清单**，帮你在出事之前把问题排查清楚。它是所有 skill 中内容最全、覆盖面最广的一个。

### 能帮你做什么

**市场合规检查清单：**
- **美国**：FCC、FDA、CPC（儿童产品）、UL认证、Prop 65（加州65号法案）、HS编码、进口商编号
- **欧盟**：CE认证、RoHS、REACH、WEEE回收指令、UKCA（英国脱欧后）、GPSR（2024年12月新规）、VAT/OSS注册、EORI号
- **日本**：PSE认证（电器安全法）、TELEC（无线设备）、日语标签要求
- **中国国内**：营业执照范围、食品经营许可证、CCC认证、化妆品备案、中文标签要求

**知识产权风险排查：**
- 商标风险：品牌名在目标市场是否已被注册（附 USPTO/EUIPO/JPO 查询工具引导）
- 专利风险：外观、功能、包装的侵权排查方向
- 版权风险：图片、文案、包装设计的原创性核查

**税务合规方案：**
- 美国：Sales Tax 经济联结阈值（各州$100K或200单）和申报逻辑
- 欧盟：VAT vs OSS 方案对比、IOSS（进口一站式）适用场景
- 英国：脱欧后 VAT 注册阈值（£85,000）
- 税务代理选择评估框架（Avalara/Taxdoo/欧税通对比）

**物流合规：**
- FBA/3PL/自发货/Dropshipping 四种模式的合规性对比
- 头程物流方式（海运整柜/拼箱/空运/班列）的时效和清关风险
- 清关文件清单（商业发票、装箱单、提单、原产地证、产品认证）
- 关税查询与合规优化建议（合理申报、自贸协定、HS编码归类优化）

**合规文档管理体系：**
- 标准化文件夹结构（产品认证/税务/知识产权/合同/产品标签/海关）
- 证书有效期跟踪表

### 使用示例

```
"我要把一款电动玩具发到美国亚马逊，帮我列出需要准备哪些认证和合规文件"
"我们品牌名叫'ProCool'，准备在欧洲注册，帮我梳理商标和合规风险"
"我在欧洲有VAT义务，帮我对比 OSS 申报和分国注册哪个更适合"
"这份中文包装要用在日本市场，帮我检查有哪些标签不合规"
"帮我生成一份我们产品出海美国的合规风险评估报告"
```

> **重要免责说明**：此 skill 提供合规框架和排查方向，但无法替代持牌律师、认证机构或税务顾问的专业意见。实际经营前，高风险决策请以专业机构的最新意见为准。
