# 各平台字符限制速查表

## Amazon

| 字段 | 字符限制 | 说明 |
|------|---------|------|
| 标题 | 200字符 | 部分类目80字符，建议控制在120以内 |
| Bullet Points | 5条 × 500字符 | 每条建议200-300字符，可读性最佳 |
| Description | 2000字符 | A+后可不填，但后台关键词有用 |
| Backend Search Terms | 250字节 | 无需标点、重复、竞品品牌名 |
| Subject Matter | 50字符 | 部分类目有 |
| Other Attributes | 50-100字符 | 因类目而异 |

## Shopee

| 规格 | 20个选项 | 颜色/尺寸等 |
