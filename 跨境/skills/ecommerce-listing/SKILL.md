# E-commerce Listing Optimization & Content Generation

Listing是电商产品的"门面"，直接决定点击率与转化率。本技能提供系统化的Listing创作与优化工作流。

## 工作流总览

Listing创作包含以下步骤：

1. 分析平台特性与受众 → 确定内容策略
2. 关键词研究与布局
3. 标题创作（含各平台字符限制）
4. 卖点/Bullet Points 创作
5. 详情页/A+页面 内容规划
6. 视觉素材brief（主图、场景图、视频脚本）
7. 多语言版本生成（跨境场景）
8. 合规检查与最终输出

## 第一步：平台特性分析

不同平台的Listing逻辑差异显著，先确定平台再创作：

### Amazon
- **标题**：200字符以内，核心关键词前置
- **Bullet Points**：5点，每点500字符以内
- **Description**：2000字符，HTML支持
- **A+ Content**：图文模块，品牌备案后可用
- **关键词**：Backend Search Terms 250字节
- **逻辑**：搜索导向，SEO权重极高

### Shopee/Lazada
- **标题**：60-120字符，简洁明了
- **描述**：支持图文混排
- **逻辑**：移动端优先，价格敏感度极高
- **特点**： emoji使用可提升亲和力

### Temu
- **标题**：简洁，突出核心卖点
- **逻辑**：平台推荐导向，低价心智
- **特点**：供货模式，前台展示由平台控制较多

### 淘宝/天猫
- **标题**：30汉字，关键词覆盖
- **详情页**：长图文为主， storytelling 重要
- **逻辑**：搜索+推荐双驱动
- **特点**：买家秀、问大家等UGC模块重要

### 抖音电商
- **标题**：短平快，突出痛点/利益点
- **详情页**：短视频化，3秒定生死
- **逻辑**：兴趣推荐，内容驱动
- **特点**：场景化展示，情绪价值

### 拼多多
- **标题**：突出价格优势和核心功能
- **逻辑**：价格敏感，团购心智
- **特点**：简洁直接，减少废话

## 第二步：关键词研究框架

关键词是Listing的流量基础，按以下步骤执行：

1. **核心词挖掘**：产品本质名称（如"wireless earbuds"）
2. **长尾词扩展**：场景、人群、功能修饰词
3. **竞品ASIN反查**：提取竞品流量词
4. **搜索建议收集**：平台下拉框、相关搜索
5. **关键词分类**：
   - 大词/流量词：搜索量大，竞争高，放标题
   - 长尾词/转化词：搜索量小，精准度高，放BP/描述
   - 场景词：使用场景描述，放A+/详情页

### 关键词布局表

```markdown
| 位置 | 关键词类型 | 数量 | 示例 |
|------|-----------|------|------|
| 标题 | 核心词+大词 | 2-3个 | Wireless Earbuds Bluetooth 5.3 |
| Bullet 1 | 核心功能词 | 1-2个 | Noise Cancelling Headphones |
| Bullet 2-4 | 长尾词 | 各1个 | Sports Earphones IPX7 Waterproof |
| 描述 | 场景词+长尾 | 3-5个 | Gym Running Workout Headset |
| Search Terms | 补充词 | 所有剩余精准词 | （仅Amazon） |
```

## 第三步：标题创作公式

### Amazon标题公式
```
[品牌] + [核心关键词] + [核心功能1] + [核心功能2] + [型号/规格] + [适用场景]
```
**示例：**
```
SoundPro Wireless Earbuds Bluetooth 5.3 with Active Noise Cancelling,
40H Playtime IPX7 Waterproof Sports Earphones for Running Gym, Black
```
**限制**：< 200字符，首字母大写，禁用促销词汇

### 淘宝标题公式
```
[营销词] + [核心关键词] + [属性词1] + [属性词2] + [场景词] + [长尾词]
```
**示例：**
```
【李佳琦推荐】无线蓝牙耳机2024新款降噪运动跑步超长续航适用苹果华为
```
**限制**：30汉字（60字符），填满为优

### 跨境多语言适配
- 欧洲站：注意德语词长、法语性数配合
- 日本站：敬体使用，细节描述偏好
- 中东站：RTL排版，文化禁忌词过滤
- 拉美站：葡萄牙语/西班牙语本地化

## 第四步：Bullet Points / 卖点创作

### Amazon 5点描述公式

**结构：大写标题 + 详细描述**

```markdown
1. 【核心卖点】一句话总结 → 展开2-3句说明，带数据支撑
2. 【功能卖点】具体功能 → 使用场景+用户收益
3. 【差异化】与竞品差异 → 技术优势或设计巧思
4. 【信任背书】品质/认证 → 消除购买顾虑
5. 【使用场景】Who/When/Where → 让用户代入想象
```

**示例：**
```markdown
* ACTIVE NOISE CANCELLING FOR IMMERSIVE SOUND — Advanced ANC technology
  reduces ambient noise by up to 35dB, letting you focus on music or calls
  even in busy environments like airplanes and coffee shops.

* 40-HOUR BATTERY WITH QUICK CHARGE — Enjoy up to 8 hours per charge and
  4 additional full charges from the case. A 10-minute quick charge delivers
  2 hours of playback — perfect for your daily commute.
```

### 国内平台卖点公式

更强调情感共鸣和场景代入：

```markdown
1. 【痛点共鸣】你是否有XXX的烦恼？→ 我们有解决方案
2. 【核心功能】技术参数 → 通俗化翻译为用户语言
3. 【场景展示】在什么场景下使用 → 画面感描述
4. 【信任证明】销量/评价/认证/明星同款
5. 【售后保障】无忧购买 → 降低决策门槛
```

## 第五步：详情页/A+内容规划

### Amazon A+ 模块规划（品牌备案后）

```markdown
## A+ Content 方案

### 模块1：品牌故事（标准比较图）
- 品牌理念1句话
- 品牌视觉：生活方式图片

### 模块2：核心卖点大图（标准单图）
- 1张场景大图 + 3个icon卖点

### 模块3：功能详解（标准四图）
- 4个功能点，每个配小图+文字

### 模块4：规格参数（标准四图/表格）
- 技术规格一览表

### 模块5：使用场景（标准多图）
- 3个使用场景图

### 模块6：品牌对比（标准比较图）
- 与竞品对比表（注意合规，不直接点名竞品）
```

### 淘宝详情页规划

```markdown
## 详情页结构（共7-10屏）

第1屏：首图/海报图 — 视觉冲击，核心卖点一句话
第2屏：痛点共鸣 — 3个痛点+解决方案
第3屏：产品全貌 — 360度展示或核心功能图
第4屏：功能详解1 — 核心功能展开
第5屏：功能详解2 — 次要功能展开
第6屏：场景展示 — 2-3个使用场景
第7屏：细节/工艺 — 质感展示，建立信任
第8屏：参数/规格 — 技术参数表格
第9屏：包装/配件 — 所见即所得
第10屏：品牌背书/售后 — 消除顾虑
```

## 第六步：视觉素材Brief

每次Listing创作同步输出视觉素材需求文档：

```markdown
## 视觉素材需求：[产品名]

### 主图（第1张 - 白底图）
- 背景：纯白（#FFFFFF）
- 主体：产品正面45度角
- 尺寸：2000x2000px（Amazon）/ 800x800px（淘宝）
- 要求：占据画面85%以上，无水印无文字（Amazon）

### 主图（第2-6张 - 场景/功能图）
- 图2：使用场景图（户外/室内）
- 图3：功能特写图（核心卖点可视化）
- 图4：尺寸/规格参考图（带生活参照物）
- 图5：包装内含物全家福
- 图6：品牌认证/品质背书图

### 视频脚本（15-30秒）
- 0-3s：痛点/钩子画面
- 3-10s：产品全貌+核心功能展示
- 10-20s：使用场景+用户体验
- 20-30s：行动号召/促销信息
```

## 第七步：多语言生成规范

跨境场景下需要多语言版本时，遵循以下原则：

1. **先完成英文母版**，再翻译到目标语言
2. **专业术语保留英文**（如Bluetooth, IPX7 Waterproof）
3. **文化适配**：
   - 德语：正式语气(Sie)，长复合词合理断行
   - 日语：です/ます体，细节控（包装内容详细列出）
   - 法语：性数一致，优雅简洁
   - 西班牙语：区分拉美/西班牙用词差异
4. **字符限制**：翻译后必须检查目标平台字符限制
5. **禁忌词过滤**：各国禁用词汇清单（见参考文件）

## 第八步：合规检查清单

Listing发布前必须通过以下检查：

- [ ] 无绝对化用语（最好、第一、100%等）— 国内平台
- [ ] 无疾病治疗宣称 — 全平台
- [ ] 无侵权品牌词/商标词 — 全平台
- [ ] 无虚假/夸大参数 — 全平台
- [ ] 符合目标市场广告法（如FDA、FTC规定）— 跨境
- [ ] 图片无水印、无边框、无促销信息 — Amazon
- [ ] 标题无特殊字符（! $ ? 等）— Amazon

## 输出格式

最终交付物为 **Listing完整方案包**，包含：
1. 关键词研究与布局表
2. 标题（各平台版本）
3. Bullet Points / 卖点描述
4. 详情页/A+内容规划
5. 视觉素材Brief
6. 多语言版本（如需要）
7. 合规检查确认
