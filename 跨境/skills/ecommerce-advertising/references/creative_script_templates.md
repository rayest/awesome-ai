# 广告素材脚本模板

## 短视频 15 秒模板

1. 0-3 秒：痛点或强场景开场。
2. 3-8 秒：展示产品核心功能或对比效果。
3. 8-12 秒：给出证据，例如真实使用场景、参数、评价摘录。
4. 12-15 秒：明确行动号召，例如领取优惠、查看详情、加入购物车。

## 图片广告 Brief

| 模块 | 要求 |
|---|---|
| 主视觉 | 产品占画面 40%-60%，背景干净 |
| 文案 | 一屏只表达一个核心利益点 |
| 证明 | 可加入评分、销量、使用前后对比，但必须真实可核验 |
| CTA | 按平台规范使用，避免夸大承诺 |

## 合规提醒

- 不使用绝对化承诺，例如“第一”“100%有效”“永久解决”。
- 不使用未经授权的品牌、名人、竞品素材。
- 医疗、健康、美容、儿童、食品等类目需额外复核平台政策。
