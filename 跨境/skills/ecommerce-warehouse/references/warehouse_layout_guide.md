# 仓库布局设计指南

## 功能区划分

| 区域 | 作用 | 设计要点 |
|---|---|---|
| 收货区 | 到货清点和质检 | 靠近入口，预留暂存位 |
| 存储区 | 正常库存存放 | 按 ABC 和周转率分区 |
| 拣货区 | 高频 SKU 拣选 | A 类 SKU 靠近打包台 |
| 打包区 | 复核、包装、贴单 | 动线短，耗材易取 |
| 发货区 | 按物流商交接 | 按渠道分笼/分区 |
| 退货区 | 退件检查和再上架 | 与正常库存隔离 |

## 动线原则

- 收货、上架、拣货、打包、发货尽量单向流动。
- 高频 SKU 放在黄金拣货区，减少行走距离。
- 退货、异常、待检商品不要混入可售库存。
- 大促前预留临时爆品区和打包扩容区。

## 布局评估指标

- 单均拣货距离。
- 每小时打包单量。
- 错发漏发率。
- 库位利用率。
- 退货处理周期。
