# E-commerce Warehouse & Inventory Management

仓储库存管理是电商运营的资金效率核心。本技能提供系统化的库存控制与仓储优化框架。

## 工作流总览

仓储管理包含以下模块：

1. 仓库布局与选址
2. 库存分类管理（ABC）
3. 安全库存与补货策略
4. 库存预警与监控
5. 滞销品处理
6. 多仓库存分配
7. 库存周转优化

## 输入信息要求

库存分析前，应尽量收集：

| 信息 | 示例 |
|------|------|
| SKU 基础信息 | SKU、类目、成本、售价、体积 |
| 销售数据 | 日销量、周销量、季节性、促销计划 |
| 供应数据 | 采购周期、生产周期、运输周期、MOQ |
| 库存数据 | 可售库存、在途库存、锁定库存、退货库存 |
| 风险要求 | 目标服务水平、可接受缺货率、资金上限 |

## 输出质量标准

- 必须区分缺货风险、滞销风险和资金占用风险。
- 补货建议应给出数量、触发条件和复核周期。
- 对新品、爆品、长尾品应使用不同库存策略。

## 第一步：仓库布局与选址

### 国内电商仓储模式选择

| 模式 | 固定成本 | 变动成本 | 时效控制 | 弹性扩容 | 适用场景 |
|------|---------|---------|---------|---------|---------|
| 平台官方仓 | 中等 | 中等 | 强 | 中等 | 品牌卖家、追求流量权重 |
| 第三方仓 | 低 | 中等 | 中等 | 强 | 中小卖家、多平台运营 |
| 自建仓 | 高 | 低 | 极强 | 弱 | 大体量、特殊品类 |

### 跨境电商仓储模式

| 模式 | 时效 | 成本 | 资金占用 | 流量权重 | 适用场景 |
|------|------|------|---------|---------|---------|
| FBA/平台仓 | 1-3天 | 高 | 高 | 高 | 标准件、稳定销量 |
| 第三方海外仓 | 2-5天 | 中 | 中 | 中 | 多平台、大件产品 |
| 自发货 | 7-30天 | 低 | 低 | 低 | 测试期、低客单价 |
| Dropshipping | 10-30天 | 极低 | 极低 | 低 | 零起步测试 |

## 第二步：库存分类管理（ABC）

### ABC分类标准

| 分类 | 销量占比 | SKU占比 | 管理策略 |
|------|---------|---------|---------|
| A类 | 70% | 10-20% | 每日监控、安全库存、优先补货 |
| B类 | 20% | 20-30% | 每周监控、正常补货 |
| C类 | 10% | 50-70% | 每月审查、清仓或淘汰 |

### ABC分类执行步骤

1. 导出过去3-6个月销售数据
2. 按SKU汇总销售额/销量
3. 降序排列，计算累计占比
4. 按70%/20%/10%阈值划分ABC
5. 制定差异化管理策略

## 第三步：安全库存与补货策略

### 安全库存计算公式

```
安全库存 = Z × σ × √L

其中：
- Z：服务水平系数（95%服务水平 Z=1.65，99% Z=2.33）
- σ：日销量标准差
- L：补货周期（天）
```

### 补货点计算

```
补货点 = 日均销量 × 补货周期 + 安全库存

经济订货量（EOQ）= √(2 × D × S / H)
其中：
- D：年需求量
- S：每次订货成本
- H：单位库存持有成本
```

## 第四步：库存预警与监控

### 预警规则配置

| 预警类型 | 触发条件 | 通知方式 | 处理时限 |
|---------|---------|---------|---------|
| 缺货预警 | 可售天数 ≤ 3天 | 即时通知 | 24h内补货 |
| 低库存预警 | 可售天数 ≤ 7天 | 每日汇总 | 48h内补货 |
| 滞销预警 | 库龄 > 90天 | 每周汇总 | 7天内处理 |
| 超储预警 | 库存 > 3个月销量 | 每月汇总 | 30天内调整 |
| 效期预警 | 保质期剩余 < 30% | 即时通知 | 立即促销 |

## 第五步：滞销品处理

### 滞销处理决策树

```
库龄 > 90天?
  ├─ 是 → 还有销售?
  │       ├─ 是 → 降价促销（8折→7折→5折）
  │       └─ 否 → 库龄 > 180天?
  │               ├─ 是 → 站外清仓/团购
  │               └─ 否 → 库龄 > 365天?
  │                       ├─ 是 → 报废/捐赠
  │                       └─ 否 → 继续观察
  └─ 否 → 正常销售
```

## 第六步：多仓库存分配

### 库存分配算法

```markdown
## 多仓分配策略

### 因素权重
- 销量预测：40%
- 物流成本：25%
- 仓储成本：20%
- 时效要求：15%

### 分配公式
仓库A分配量 = 总需求 × (仓库A销量占比×0.4 + 仓库A成本优势×0.25 + 仓库A时效优势×0.15)

### 动态调拨触发条件
- 仓库A可售天数 < 7天 且 仓库B可售天数 > 30天
- 调拨成本 < 缺货损失预估
```

## 第七步：库存周转优化

### 周转率提升策略

| 策略 | 适用场景 | 预期效果 |
|------|---------|---------|
| 精准预测 | 数据积累充足 | 周转天数降低20-30% |
| 小批量多频次 | 高价值、易过时品类 | 滞销率降低50% |
| 预售模式 | 定制化、高客单价 | 库存资金占用降低70% |
| 供应商VMI | 战略合作供应商 | 安全库存降低30% |
