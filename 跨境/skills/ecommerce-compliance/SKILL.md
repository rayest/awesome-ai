# E-commerce Compliance, Tax & Logistics

合规是电商经营的底线，跨境场景尤其复杂。本技能提供系统化的合规与物流工作流。

## 工作流总览

合规与物流管理包含以下模块：

1. 目标市场合规要求识别
2. 产品认证规划
3. 知识产权风险排查
4. 税务合规方案
5. 物流方案设计
6. 海关与清关
7. 退换货物流处理
8. 合规文档管理

## 第一步：目标市场合规要求

### 主要市场合规框架

#### 美国市场
```markdown
## 美国市场合规清单

### 产品认证
- [ ] FCC — 电子设备电磁兼容
- [ ] FDA — 食品/药品/化妆品/医疗器械
- [ ] CPC — 儿童产品证书（12岁以下）
- [ ] UL/ETL — 电器安全认证（非强制但推荐）
- [ ] DOE/CEC — 能效认证
- [ ] Prop 65 — 加州65号法案警告（含特定化学物质）

### 标签要求
- [ ] 原产地标签（Made in China）
- [ ] 英语标签
- [ ] 安全警告标签（如适用）
- [ ] 成分/材料标签（如适用）

### 进口要求
- [ ] HS编码确认
- [ ] 关税率查询
- [ ] 进口商编号（EIN/IRS Number）
- [ ] 产品责任保险（建议）
```
