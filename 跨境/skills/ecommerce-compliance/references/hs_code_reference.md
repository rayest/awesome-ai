# HS 编码确认参考

## 基础流程

1. 明确产品材质、用途、结构和工作原理。
2. 查询目标国家或地区海关编码库。
3. 对比相似产品的归类案例。
4. 与报关行或专业机构确认。
5. 保存确认记录、产品说明、图片和归类依据。

## 常见风险

- 只按商品名称归类，忽略材质和用途。
- 同一产品在不同市场编码不同。
- 低报、错报导致清关延误、罚款或补税。
- 组合套装未拆分主成分或主要功能。
