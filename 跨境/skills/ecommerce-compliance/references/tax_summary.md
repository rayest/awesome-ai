# 各市场税务要点

## 美国销售税

### 经济联结阈值（Economic Nexus）
| 州 | 销售额阈值 | 订单量阈值 |
|----|----------|----------|
| 加利福尼亚 | $500,000 | 无 |
| 纽约 | $500,000 | 100笔 |
| 德克萨斯 | $500,000 | 无 |
| 佛罗里达 | $100,000 | 无 |
| 伊利诺伊 | $100,000 | 无 |

### 注意事项
- Amazon代收代缴大部分州销售税
- 独立站需自行管理
- 建议使用Avalara等自动化工具

## 欧盟VAT

### 税率参考
| 国家 | 标准税率 | 低税率 |
|------|---------|--------|
| 德国 | 19% | 7% |
| 法国 | 20% | 5.5% |
| 意大利 | 22% | 4-10% |
| 西班牙 | 21% | 4-10% |
| 荷兰 | 21% | 9% |

### OSS（One-Stop-Shop）
- 2021年7月起生效
- 可在注册国一站式申报所有欧盟销售
- 阈值取消（此前各国€35K-€100K）

### IOSS
- 适用于B2C进口≤€150的货物
- 可提前收取VAT，加速清关

## 英国VAT

### 关键要点
- 脱欧后独立体系
- 注册阈值：£85,000
- 标准税率：20%
- 可使用HMRC在线系统申报

## 日本消费税（JCT）

### 关键要点
- 注册阈值：年销售额¥10,000,000
- 标准税率：10%（消费税8%+地方消费税2%）
- 2023年起Amazon代扣代缴

## 中国税务

### 跨境电商综合税
- 关税：0%（限值内）
- 增值税：70%征收（13%×70%=9.1%）
- 消费税：70%征收（如适用）
- 限值：单笔¥5,000，年度¥26,000

### 国内电商
- 增值税：小规模纳税人1-3%，一般纳税人6-13%
- 企业所得税：25%（小微企业有优惠）
