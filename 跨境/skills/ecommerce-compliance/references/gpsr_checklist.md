# 欧盟 GPSR 初筛清单

GPSR 适用于欧盟一般消费品安全要求。实际合规应以官方法规、欧盟授权代表和专业机构意见为准。

## 基础检查

- [ ] 产品有明确制造商或责任主体信息。
- [ ] 有欧盟境内责任人或授权代表信息。
- [ ] 标签、说明书和安全警示适配目标销售国家语言。
- [ ] 风险评估覆盖可预见误用场景。
- [ ] 可追溯批次、型号、SKU、生产日期或批号。
- [ ] 保存测试、认证、供应商和质量文件。
- [ ] 建立召回和消费者投诉处理流程。

## 高风险提示

- 儿童用品、带电产品、食品接触材料、美妆个护、运动防护用品需更严格复核。
- 不确定是否适用时，先按更高风险等级处理。
