# 供应商验厂清单

## 基础资质

- [ ] 营业执照与经营范围匹配。
- [ ] 生产地址、仓库地址和开票主体一致或可解释。
- [ ] 有稳定联系人和售后负责人。
- [ ] 能提供过往合作案例或平台供货记录。

## 生产能力

- [ ] 核心设备数量和状态。
- [ ] 月产能与旺季产能。
- [ ] 原材料来源和备货周期。
- [ ] 交期承诺和延期补救机制。

## 质量体系

- [ ] 来料检验、过程检验、出货检验流程。
- [ ] 不良品处理记录。
- [ ] 抽检标准和 AQL 规则。
- [ ] 可提供样品和小批量测试。

## 合规与外贸能力

- [ ] 是否有出口经验。
- [ ] 是否能提供 CE/FCC/FDA/CPC/RoHS 等相关资料。
- [ ] 包装、标签、说明书是否支持目标市场语言。
