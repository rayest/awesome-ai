# E-commerce Data Analytics & Reporting

数据驱动决策是电商运营的核心竞争力。本技能提供系统化的数据分析框架与报表体系。

## 工作流总览

电商数据分析包含以下模块：

1. 确定分析目标与数据源
2. 核心指标体系建设
3. 日常销售报表
4. 流量与转化分析
5. 利润与财务分析
6. 库存分析
7. 竞品监控
8. 自动化报表搭建

## 第一步：确定分析目标

不同分析目标对应不同指标体系：

| 分析目标 | 核心关注 | 关键指标 | 分析频率 |
|---------|---------|---------|---------|
| 销售健康度 | 整体表现 | GMV、订单量、客单价、环比 | 日报 |
| 流量效率 | 获客能力 | UV、PV、CTR、CPC | 日报 |
| 转化优化 | 成交能力 | CVR、加购率、结账率 | 周报 |
| 盈利能力 | 赚钱能力 | 毛利率、净利率、ROAS | 周报 |
| 库存周转 | 资金效率 | 周转天数、滞销率、缺货率 | 周报 |
| 竞品动态 | 竞争态势 | 竞品价格、销量、新品 | 月报 |
| 用户价值 | 长期健康 | 复购率、LTV、NPS | 月报 |

## 第二步：核心指标体系

### 2.1 通用指标体系（跨境+国内）

```markdown
## 电商核心指标金字塔

### 第一层：业绩指标（结果）
- GMV（总成交额）= UV × 转化率 × 客单价
- 订单量
- 实收金额（扣除退款）

### 第二层：流量指标（入口）
- 曝光量（Impressions）
- 点击量（Clicks）/ 访客数（UV）
- CTR（点击率）= 点击/曝光
- CPC（单次点击成本）= 花费/点击
- 流量来源占比（搜索/推荐/广告/直接访问）

### 第三层：转化指标（效率）
- 加购率 = 加购人数/访客数
- 收藏/关注率
- 结账率 = 结账人数/加购人数
- 支付转化率（CVR）= 支付订单数/访客数
- 客单价（AOV）= GMV/订单数
- 连带率 = 总件数/订单数

### 第四层：盈利指标（质量）
- 毛利率 = (售价-成本)/售价
- 净利率 = 净利润/GMV
- ROAS = 广告销售额/广告花费
- ACOS = 广告花费/广告销售额（Amazon）
- CPA（单客获取成本）= 营销花费/新客数
- LTV（客户生命周期价值）= 客单价×复购次数×生命周期

### 第五层：运营效率指标（健康）
- 库存周转天数 = 平均库存/日均成本
- 缺货率 = 缺货SKU/总SKU
- 滞销率 = 滞销SKU/总SKU
- 退货率 = 退货订单/总订单
- 差评率 = 差评数/总评价数
- 响应时效 = 客服平均响应时间
```

### 2.2 平台特有指标

**Amazon特有：**
- BSR（Best Seller Rank）
- Buy Box占比
- Session Percentage（转化率平台版）
- 账户健康分
- IPI（库存绩效指标）

**国内平台特有：**
- DSR/店铺评分（淘宝）
- 体验分（抖音）
- 风向标（京东）
- 店铺层级（影响流量分配）
