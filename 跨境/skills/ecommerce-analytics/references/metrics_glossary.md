# 电商指标术语表

| 指标 | 公式 | 用途 |
|---|---|---|
| GMV | 成交金额总和 | 衡量交易规模 |
| 净销售额 | GMV - 退款 - 取消 | 衡量实际收入 |
| 客单价 AOV | 销售额 / 订单数 | 判断用户购买强度 |
| 转化率 CVR | 订单数 / 访客数 | 衡量页面与流量质量 |
| 点击率 CTR | 点击数 / 展示数 | 衡量素材和标题吸引力 |
| ROAS | 广告销售额 / 广告花费 | 衡量广告产出 |
| ACOS | 广告花费 / 广告销售额 | Amazon 常用广告成本率 |
| 毛利率 | 毛利 / 销售额 | 判断单品盈利能力 |
| 库存周转天数 | 平均库存 / 日均销量 | 衡量库存资金效率 |
| 退款率 | 退款订单 / 总订单 | 衡量质量、描述和服务风险 |

## 使用原则

- 先统一口径，再比较数据。
- 区分 GMV、支付金额、发货金额和净销售额。
- 广告复盘同时看 ROAS、ACOS、订单量和毛利，不只看单一指标。
