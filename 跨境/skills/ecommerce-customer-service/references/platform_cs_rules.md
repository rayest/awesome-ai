# 平台客服规则速查

| 平台 | 重点规则 | 风险点 |
|---|---|---|
| Amazon | 站内信合规、不得诱导好评 | 索评话术、外链、补偿换好评 |
| Shopee | 响应时效、聊聊服务指标 | 超时回复、争议处理慢 |
| Temu | 平台履约和售后规则强约束 | 私下引导、证据不足 |
| 淘宝/天猫 | 消费者权益、平台介入 | 承诺不一致、辱骂或推责 |
| 京东 | 时效、发票、售后服务 | 超时处理、服务单证据不足 |
| 抖音/拼多多 | 体验分、退款和物流表现 | 低价高纠纷、虚假发货 |

## 通用底线

- 不引导用户离开平台交易。
- 不要求用户删除差评后才处理问题。
- 不承诺平台规则之外的补偿。
- 关键沟通保留在平台可追溯渠道内。
