# E-commerce Logistics & Fulfillment

物流配送是电商履约体验的核心环节。本技能提供系统化的物流方案设计与成本优化框架。

## 工作流总览

物流管理包含以下模块：

1. 物流模式选择
2. 头程物流方案（跨境）
3. 尾程配送网络
4. 物流成本核算
5. 智能路由推荐
6. 时效监控与异常处理
7. 物流绩效分析

## 输入信息要求

物流方案设计前，应尽量确认以下信息：

| 信息 | 示例 |
|------|------|
| 商品属性 | 重量、体积、是否易碎、是否带电/带液/带磁 |
| 目的地 | 国家、州/省、是否偏远地区 |
| 履约模式 | FBA、海外仓、自发货、国内快递 |
| 时效承诺 | 平台要求、店铺承诺、用户可接受时间 |
| 成本上限 | 物流费占售价或GMV的目标比例 |

## 输出质量标准

- 至少给出 2-3 个可选方案，并说明适用场景。
- 必须把成本、时效、稳定性、清关和售后风险分开评估。
- 跨境方案必须提示清关、税费、禁限运和退货成本不确定性。

## 第一步：物流模式选择

### 国内电商物流方案

| 方案 | 时效 | 成本（1kg） | 适用场景 | 优势 |
|------|------|------------|---------|------|
| 通达系 | 2-4天 | ¥3-5 | 拼多多/淘宝低价商品 | 成本低、覆盖广 |
| 顺丰标快 | 1-2天 | ¥12-18 | 高价值/急件 | 时效稳定、服务好 |
| 京东物流 | 1-2天 | ¥10-15 | 京东平台/自营 | 体验佳、逆向强 |
| 同城即时 | 1-4小时 | ¥15-30 | 生鲜/急用 | 极致时效 |

## 第四步：物流成本核算

### 全链路成本模型

## 物流全链路成本

### 国内电商

```text
单件物流成本 = 快递费 + 包装费 + 面单费 + 操作费

示例：
- 快递费（通达系）：¥3.5
- 包装费（纸箱+填充）：¥1.2
- 面单费：¥0.3
- 操作费（拣货+打包）：¥0.8
- 单件总成本：¥5.8
```

### 跨境电商（FBA模式）

```text
单件物流成本 = 头程分摊 + FBA配送费 + FBA仓储费 + 退货处理费

示例：
- 头程海运分摊：$1.2
- FBA配送费：$3.5
- FBA仓储费（月均）：$0.5
- 退货处理费：$0.3
- 单件总成本：$5.5
```

### 跨境电商（自发货模式）

```text
单件物流成本 = 国内快递 + 国际运费 + 清关费 + 末端配送

示例：
- 国内到集货仓：¥3
- 国际专线运费：$4
- 清关费分摊：$0.5
- 末端配送：$2
- 单件总成本：$6.5
```

## 第五步：智能路由推荐

### 路由决策矩阵

| 因素 | 权重 | 评估标准 |
|------|------|---------|
| 时效要求 | 30% | 承诺送达日期 vs 各方案时效 |
| 成本预算 | 25% | 物流成本占售价比例 |
| 商品属性 | 20% | 重量、体积、易碎、温控 |
| 目的地 | 15% | 偏远程度、清关难度 |
| 服务等级 | 10% | 追踪、保险、签收确认 |

### 路由推荐逻辑

```
输入：订单参数（重量、目的地、时效要求、商品价值）
  ↓
筛选：排除不满足时效要求的方案
  ↓
评分：各方案按权重打分
  ↓
推荐：Top 3 方案排序输出
  ↓
决策：人工确认或自动选择（按规则）
```

## 第六步：时效监控与异常处理

### 物流节点追踪

```markdown
## 标准物流节点

国内快递：
[下单] → [揽收] → [转运中心] → [目的地网点] → [派送] → [签收]

跨境专线：
[下单] → [国内揽收] → [集货仓] → [出口清关] → [国际运输] → [进口清关] → [末端配送] → [签收]

FBA头程：
[下单] → [工厂提货] → [国内仓] → [出口报关] → [海运/空运] → [目的港] → [清关] → [FBA入仓]
```

### 异常处理SOP

| 异常类型 | 判断标准 | 处理方案 | 时效 |
|---------|---------|---------|------|
| 未揽收 | 48h无更新 | 催促发货/换单 | 4h |
| 运输停滞 | 72h无更新 | 查询物流商+安抚客户 | 12h |
| 清关延误 | 超预计3天 | 提供补充资料/换行清关 | 24h |
| 包裹丢失 | 物流确认 | 补发/退款+索赔 | 24h |
| 破损拒收 | 签收异常 | 拍照取证+补发/退款 | 24h |

## 第七步：物流绩效分析

### 核心KPI

| 指标 | 计算公式 | 健康阈值 | 优化方向 |
|------|---------|---------|---------|
| 平均配送时效 | 总天数/订单数 | 国内<3天/跨境<10天 | 优化路由 |
| 准时送达率 | 准时单/总单 | ≥95% | 选择更稳渠道 |
| 物流成本占比 | 物流费/GMV | 国内<8%/跨境<15% | 议价/集采 |
| 破损/丢失率 | 异常单/总单 | ≤0.5% | 加强包装 |
| 退货物流成本 | 退货费/总物流费 | ≤10% | 优化退货方案 |
