# 供应商评估清单

## 基础信息核验
- [ ] 营业执照（经营范围、注册资本）
- [ ] 工厂实地/视频验厂
- [ ] 生产年限与规模
- [ ] 主要客户案例（可核实的）
- [ ] 阿里诚信通年限/信保额度

## 生产能力评估
- [ ] 日/月产能
- [ ] 打样周期
- [ ] 大货交期
- [ ] 产能弹性（旺季能否扩产）
- [ ] 生产自动化程度

## 品质管理
- [ ] ISO9001认证
- [ ] 品控流程（来料检-过程检-出货检）
- [ ] 良品率数据
- [ ] 不良品处理方案
- [ ] 是否有第三方质检报告

## 认证与合规
- [ ] 已有产品认证清单
- [ ] 配合新认证意愿
- [ ] 环保合规（如ROHS、REACH）
- [ ] 社会责任（BSCI/Sedex）

## 商务条件
- [ ] MOQ（最小起订量）
- [ ] 报价有效期
- [ ] 付款方式（预付比例、账期）
- [ ] 包装方式与费用
- [ ] 物流配合度

## 沟通与响应
- [ ] 业务员专业度
- [ ] 响应时效
- [ ] 英语/其他语言能力
- [ ] 问题处理态度

## 评分卡
| 维度 | 权重 | 得分(1-5) | 加权分 |
|------|------|----------|--------|
| 生产能力 | 20% | | |
| 品质管理 | 25% | | |
| 认证合规 | 15% | | |
| 商务条件 | 20% | | |
| 沟通响应 | 20% | | |
