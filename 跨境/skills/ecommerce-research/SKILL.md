# E-commerce Market Research & Product Sourcing

市场调研与选品决策是电商运营的第一步。本技能提供系统化的调研框架，同时覆盖跨境与国内双场景。

## 工作流总览

选品决策包含以下步骤：

1. 确定目标市场与平台 → 选择调研框架
2. 收集市场数据（搜索量、趋势、竞品）
3. 竞品分析与差异化定位
4. 供应链初筛与成本估算
5. 利润测算与定价策略
6. 输出选品决策报告

## 输入信息要求

开始选品前，优先向用户确认以下信息；如果缺失，应在报告中标注假设条件，而不是直接给确定结论：

| 信息 | 示例 |
|------|------|
| 目标平台 | Amazon US / Shopee MY / 抖音 / 拼多多 |
| 目标品类 | 宠物用品、家居收纳、个护工具 |
| 预算与资金周期 | 首批采购预算、可承受库存周期 |
| 供应链约束 | 是否有工厂、是否可做认证、MOQ上限 |
| 风险偏好 | 稳健跟卖、差异化新品、高毛利测试 |

## 输出质量标准

- 必须给出 GO / TEST / NO-GO 结论。
- 必须列出关键假设和缺失数据。
- 必须同时评估需求、竞争、利润、供应链和合规风险。
- 不允许只给“可以做/不建议做”的泛泛建议。

## 第一步：确定调研模式

根据用户提到的平台选择对应框架：

**跨境平台：**
- Amazon → 关注BSR排名、Review数量、关键词搜索量、FBA费用
- Shopee/Lazada → 关注站点差异、本地化竞争、物流时效
- Temu → 关注平台补贴策略、低价竞争、供货价模式
- 独立站（Shopify等）→ 关注Facebook/Google广告竞争度、受众规模
- TikTok Shop → 关注短视频趋势、达人合作、冲动消费品类

**国内平台：**
- 淘宝/天猫 → 关注搜索人气、直通车竞价、竞品销量
- 京东 → 关注自营竞争、物流体验评分、品类门槛
- 抖音电商 → 关注内容热度、直播转化率、达人带货数据
- 拼多多 → 关注价格带竞争、团购机制、工厂货源

## 第二步：数据收集框架

对每个候选品类，收集以下核心指标：

### 2.1 市场需求指标

| 指标 | 数据来源 | 判断标准 |
|------|---------|---------|
| 搜索量趋势 | Google Trends/百度指数/平台后台 | 上升趋势或稳定，非季节性过强 |
| 关键词竞争度 | Jungle Scout/卖家精灵/生意参谋 | 中等竞争为佳，避免红海 |
| 市场容量 | 平台榜单/行业报告 | 足够大以支撑盈利，足够小以避开巨头 |
| 增长率 | 行业报告/趋势工具 | 年增长率 > 15% 为优 |

### 2.2 竞争格局指标

| 指标 | 数据来源 | 判断标准 |
|------|---------|---------|
| 头部卖家集中度 | 平台前台/第三方工具 | 头部占比 < 40%，分散为佳 |
| Review平均数量 | 平台前台 | 中等数量表示有机会进入 |
| 新品成功率 | 竞品追踪 | 近3个月新品占比 > 10% |
| 差异化空间 | 产品分析 | 功能/设计/服务有可改进点 |

### 2.3 利润指标

| 指标 | 计算方式 | 判断标准 |
|------|---------|---------|
| 毛利率 | (售价-成本)/售价 | > 30% 为可接受 |
| 净利率 | 扣除所有费用后 | > 15% 为优 |
| 客单价 | 平台数据 | 跨境>$20，国内>¥30 较安全 |
| 复购潜力 | 产品属性判断 | 消耗品/配件复购率高 |

## 第三步：竞品分析模板

对每个核心竞品，输出以下分析：

```markdown
## 竞品分析：[产品名称]

### 基础信息
- 品牌/店铺：
- 价格区间：
- 评分/Reviews：
- 上架时间：
- 月销量估算：

### 核心卖点
- [卖点1]
- [卖点2]

### 用户评价关键词
- 好评关键词：[关键词列表]
- 差评关键词：[关键词列表] ← 改进机会

### 差异化机会
- [机会1：具体描述]
- [机会2：具体描述]
```

## 第四步：选品评分卡

使用加权评分卡对候选产品进行量化评估：

```markdown
## 选品评分卡：[产品名]

| 维度 | 权重 | 得分(1-5) | 加权分 | 备注 |
|------|------|----------|--------|------|
| 市场需求 | 20% | | | |
| 竞争强度 | 15% | | | 分数越高竞争越小 |
| 利润空间 | 20% | | | |
| 供应链难度 | 15% | | | 分数越高越容易 |
| 物流适配 | 15% | | | 跨境需特别注意 |
| 合规风险 | 15% | | | 分数越高风险越小 |
| **总分** | 100% | | **X.XX** | |

### 结论
- 建议：GO / CONDITIONAL / NO-GO
- 关键风险：[如有]
- 下一步行动：[具体行动]
```

## 第五步：供应链初筛

对通过评分的产品，进行供应链可行性初筛：

1. **1688/阿里巴巴** 搜索供应商，记录：MOQ、报价、交期、认证情况
2. **产业带定位**：确认产品主要产区（如深圳电子、义乌小商品、佛山家具）
3. **样品评估**：确定样品费、打样周期
4. **认证要求**：
   - 跨境：CE/FCC/FDA/CPC/ROHS 等目标市场认证
   - 国内：3C认证、质检报告等

## 第六步：定价策略

根据平台特性选择定价模型：

**跨境电商定价公式：**
```
售价 = (采购成本 + 头程运费) / (1 - 平台佣金% - 毛利率目标%)
        + FBA/尾程费用 + 其他杂费
```

**国内电商定价公式：**
```
售价 = 采购成本 / (1 - 平台扣点% - 推广费率% - 毛利率目标%)
        + 运费 + 包装费
```

**定价策略类型：**
- 渗透定价：低价入市抢排名，后逐步提价
- 撇脂定价：高价定位高端，逐步下探
- 竞争定价：对标竞品±5%
- 心理定价：以9/8结尾，或整数定位

## 输出格式

最终交付物为 **选品决策报告**，包含：
1. 执行摘要（GO/NO-GO结论）
2. 市场概况与趋势
3. 竞品分析汇总
4. 选品评分卡
5. 供应链初步方案
6. 定价建议
7. 风险与缓解措施
8. 下一步行动计划
