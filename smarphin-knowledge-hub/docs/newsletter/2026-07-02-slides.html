Research Findings

Agent 研究的共同方向：约束、验证、记忆

邮件事实

arXiv 邮件中，《Making Failure Safe》把 LLM 输出限制为 typed JSON collector configs；《Mnemosyne》把 AI workflow 动作视为未可信提案，需通过确定性约束集；记忆架构论文显示 persistent private notebook 比单纯通道容量更重要。

分析：生产级 Agent 的核心不再是“让模型更自由”，而是让生成、准入、执行、修复和记忆各司其职。
