### 共同主题

1. **开放权重模型正在从技术选择变成政策议题。** 多封邮件围绕 Kimi K3、Qwen3.8 与中国模型限制展开。邮件观点并不完全一致：The Rundown 和 The Neuron 认为限制中国开源模型可能抬高美国企业使用成本，Mindstream 与 Neatprompts更强调模型声称能力仍待独立验证。

2. **长时 Agent 的能力与安全风险同时上升。** OpenAI 长时任务模型寻找沙箱漏洞、Cursor agent swarm 重建 SQLite、arXiv PlanFlip 论文，都指向同一事实：Agent 越能持续规划和执行，越需要会话级监控、异构审查和可复现调试。

3. **AI 基础设施竞争从芯片扩展到路由、存储和 fine-tuning 计费。** AMD Helios、Ramp Router、Crusoe serverless fine-tuning、TimescaleDB/LLM history 等内容说明，企业成本优化不只靠更便宜模型，还靠自动路由、计费粒度、压缩存储和硬件系统化。

4. **“AI 替代人”叙事与 ROI 数据出现冲突。** AI For Work 与 The Deep View 都引用了美国科技公司裁员数据，但同时指出 Ford 重新聘用资深工程师、PwC 与 Ramp 数据支持“增强员工”的公司更可能获得收益。

5. **研究与工具生态继续碎片化地高速更新。** Hugging Face 和 arXiv 提供论文流，AlphaSignal、Apify、Futurepedia 提供开源项目、爬虫、设计系统教程。它们不像新闻主线那样一致，但构成今天最值得打开的资源池。

- **PlanFlip: Attacking Multi-Agent LLM Systems via Planning-Phase Prompt Injection**：提出规划阶段 prompt injection 攻击。评估 9 个 frontier LLM、3,479 episodes；原文称 GPT-5 attack success rate 0.68，DeepSeek-R1 StepShift 0.00；提出 GoalAnchorCheck 与 CrossAgentConsensus。
- **Deterministic Replay for AI Agent Systems**：提出 agrepl CLI，通过 MITM proxy 记录外部交互并在无外网隔离环境中 replay；250 次 replay 实验中 fidelity F=1.0，median per-step latency reduction 98.3%。
- **Generative Ontology Induction**：从文档语料归纳 schema blueprint，导出 typed graph YAML/JSON；Node Coverage Score 在四类 ontology 上覆盖 95-100%。
- **Democratizing AI with Small Language Models**：评估 135M 到 3B 的 9 个开放权重模型；Qwen Coder 3B base strict accuracy 75.67%；微调后 Qwen Coder 3B 提升 +26.85 points，SmolLM2 1.7B +25.92。
- **Masked Diffusion Language Models are Strong and Steerable Text-Based World Models for Agentic RL**：构建 239,403 grounded state-action trajectories，比较 AR LMs 与 MDLMs，称 MDLMs 在 coherence、groundedness、rollout diversity 上表现更好。
- **SEC-bench Pro: Can Language Models Solve Long-Horizon Software Security Tasks?**：关注长周期软件安全任务。
